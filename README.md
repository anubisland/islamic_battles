# Islamic Battles Atlas

موسوعة تفاعلية عربية عن 15 معركة إسلامية كبرى، مبنية من الملفات المصدرية
المرفقة. الشاشة الرئيسية تعرض كارت مقدمة ثم كروت المعارك، وكل كارت يفتح
دراسة تفصيلية تشمل الأسباب، الأحداث، الخطة، الشخصيات، النتائج، الدروس،
نص الراوي، وخريطة تكتيكية مرئية.

## التشغيل

افتح `index.html` في المتصفح مباشرة. لا توجد خطوة بناء ولا يحتاج المشروع إلى
خادم محلي.

يمكن فتح معركة مباشرة بهذه الصيغة:

```text
index.html?battle=13
```

## الملفات

- `index.html` - هيكل الشاشة الرئيسية وشاشة التفاصيل.
- `styles.css` - تصميم نهاري واضح ومتجاوب.
- `script.js` - الكروت، البحث، التفاصيل، الخرائط SVG، والراوي الصوتي.
- `data.js` - بيانات المعارك والخرائط النصية المستخرجة من ملفات المصدر.
- `output/playwright/` - لقطات فحص بصري محلية.
